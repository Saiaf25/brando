Tickets

AURA-001 Critical

Homepage canonical conflict and sitemap duplication

The root URL canonicalises to /home, making the most valuable page on the site non-indexable. The sitemap also lists both URLs as separate entries.

Surfaces: Homepage template, sitemap generator Audit rows: 27, 40, 43, 54
AURA-002 Critical

Apex domain redirect chain crosses servers and protocols

The bare domain aurapharma.org is parked on Squarespace and 301-redirects to HTTP before HTTPS upgrades it. Two-hop chain on the wrong host.

Surfaces: DNS records Audit rows: 32, 34, 61
AURA-003 Critical

Sitewide 404s on the footer social-link block

Four footer icons link to /website/social/<platform> stubs that all return 404. 54 internal inlinks per stub. 200+ broken link relations across the crawl.

Surfaces: Sitewide footer template Audit rows: 21, 29
AURA-004 Critical

Missing H1 tags and broken heading hierarchy

Zero H1 tags on Home, About, Contact, FAQ, Shop. Decorative H3s for step numbers on the homepage. Inconsistent ALL-CAPS vs Title Case product names.

Surfaces: 5 page templates + product data layer Audit row: 65
AURA-005 Critical

Implement JSON-LD structured data sitewide

Zero structured data on every page. Missing Organization, Product, FAQPage, BreadcrumbList, ItemList, WebSite + SearchAction. Includes per-template reference JSON-LD blocks ready to adapt.

Surfaces: Base template + per-template overrides Audit rows: 66, 67, 68
AURA-006 Critical

Noindex internal search results

The /website/search endpoint and every parameter URL it produces are indexable. No meta robots tag. Page is thin (0 H1, no description). Google guidance since 2007: noindex required.

Surfaces: Search-results template Audit row: 38

Recommended ship order

  1. AURA-001 first. Every other on-page recommendation depends on the homepage being indexable.
  2. AURA-003 second. Lowest effort, removes the worst trust signal on the page.
  3. AURA-002 in parallel with the above. DNS change owned by a different person, no template work.
  4. AURA-006 next. One-line template change. Ship before AURA-005 so structured data does not amplify search-result discovery.
  5. AURA-004 next. Template-level fix, ships once.
  6. AURA-005 last and incrementally. Start with sitewide Organization, hold Product schema until real pricing replaces the 1.00 QAR placeholders.
